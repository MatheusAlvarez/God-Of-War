## 💻 Sobre o projeto

https://matheusalvarez.github.io/God-Of-War/

Criei um projeto incrível em HTML, CSS e JS que traz a vibe épica do jogo God of War Ragnarok para o mundo web. O menu, totalmente inspirado no design do jogo, oferece uma experiência imersiva aos visitantes.

Esse projeto é uma homenagem ao God of War Ragnarok, combinando técnica e estilo para criar um menu web que reflete a grandiosidade e a estética do jogo. Curta a jornada navegando por este menu inspirado em uma das sagas mais épicas dos games
